Instructions for 'tsp_group14'

To Compile, run on command line:
  make tsp_group14

To Run, run on command line:
  tsp_group14 <filename.txt>
  (ex. tsp_main data.txt)

To Validate output, run on command line:
  cat <filename.tour>

To Remove object files and executable, run on command line:
  make cleantsp

To Create a zip file
  make zip
