General instructions:
  - unzip all contents into common directory on any Flip server
  - generate an import file, such as 'data.txt'

Instructions for 'group14_tsp'

To Compile, run on command line:
  make group14_tsp

To Run, run on command line:
  group14_tsp <filename.txt>

To Validate output, run on command line:
  cat <filename.txt.tour>

To Remove object files and executable, run on command line:
  make cleantsp

To Create a zip file
  make zip
